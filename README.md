# 🧭 Pokédex Web App

Una aplicación web full stack inspirada en la Pokédex de Pokémon, donde los usuarios pueden buscar Pokémon, guardarlos en favoritos y crear equipos personalizados.

---

## 🚀 Tecnologías utilizadas

### Frontend

* React
* React Router
* Axios
* CSS / Tailwind

### Backend

* Node.js
* Express
* JWT (Autenticación)

### Base de datos

* MongoDB + Mongoose

---

## 📦 Estructura del proyecto

Este proyecto está dividido en dos repositorios:

### 🔹 Frontend

```
pokedex-frontend/
```

### 🔹 Backend

```
pokedex-backend/
```

---

## 🔐 Funcionalidades principales (MVP)

* Registro de usuarios
* Login con autenticación JWT
* Búsqueda de Pokémon (API externa)
* Ver detalles de Pokémon
* Guardar Pokémon en favoritos
* Crear equipos personalizados

---

## 🌐 API externa

Este proyecto utiliza la API pública de Pokémon:

👉 https://pokeapi.co

---

## 📡 Endpoints principales

### Auth

* POST /auth/register
* POST /auth/login

### Favorites

* POST /favorites
* GET /favorites

### Teams

* POST /teams
* GET /teams
