# 🧠 Backend - Pokédex API

API REST desarrollada con Node.js, Express y MongoDB para gestionar autenticación de usuarios y favoritos.

## 🚀 Tecnologías

- Node.js
- Express
- MongoDB (Mongoose)
- JWT (autenticación)
- bcrypt (hash de contraseñas)

## 🔐 Funcionalidades

- Registro de usuario
- Login con JWT
- Protección de rutas
- Guardado de favoritos

```bash
git clone <repo>
cd backend
npm install
